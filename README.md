All scripts for image processing and statistical analysis related to "Disrupted Perivascular MRI Signals Associate with Alzheimer’s Pathology in Down Syndrome "

Image processing directory contains all image processing scripts for DTI-ALPS, Choroid Plexus Free Water Fraction, White Matter Lesion parcellation, Tau-PET, regional amyloid-PET and Centiloids.
All image processing pipelines are written to be run on in a SLURM environment. Images downloaded from ABC-DS study on LONI
DTI-ALPS pipeline is a modified version of: https://github.com/gbarisano/alps

Analysis scripts take derived neuroimaging measures, demographic and cognitive data (available on LONI) and run statistical models presented in the manuscript
